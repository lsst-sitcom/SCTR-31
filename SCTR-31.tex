\subsection{Objectives}
\label{sect:objectives}

 The objective of this test plan is two-fold:

\begin{itemize}
\tightlist
\item
  To re-verify M2 software after M2 delivery to the summit and SAL
  version upgrade to 4.0. Make sure all events, and telemetry are
  recorded in the summit EFD.~
\item
  Run M2 performance tests. Make sure performance requirements in
  \citeds{LTS-146} are met.
\end{itemize}

{The objective of this test plan is to re-verify the functional
requirements of the M2 Assembly after shipment from the vendor's
facility to the summit facility, as defined in \citeds{LTS-146} and \citeds{LTS-162}. This
test plan consists of two test cycles:~}

\begin{itemize}
\tightlist
\item
  {A preliminary functional test cycle to initially re-verify the M2
  software after delivery to the summit and to prepare~}{for SAL 4.0
  integration}
\item
  {A formal test cycle that includes the integration of M2 with SAL 4.0,
  functional re-verification of the hardware and software, and a
  verification of M2 hazard mitigations.}
\end{itemize}

  Test Plan
